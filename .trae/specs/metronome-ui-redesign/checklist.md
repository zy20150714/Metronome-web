- [x] 主界面设计是否超级超级无敌，非常非常美观
- [x] 视觉效果是否流畅、自然，达到专业级水准
- [x] 应用是否在任何设备上正常显示

- [x] 音符设置术语是否专业、准确
- [x] 音符选项是否符合音乐标准
- [x] 应用是否在主流浏览器中正常运行

- [x] 应用是否运行稳定，没有崩溃或功能异常
