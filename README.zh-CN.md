# 节拍器应用

一款使用 React、TypeScript 和 Tailwind CSS 构建的专业节拍器应用。这款应用为音乐家和音乐学生提供了现代化、响应式的界面，包含全面的功能。

## 🌐 语言
- [English](README.md) | [中文](README.zh-CN.md)

## 🎯 功能

### 核心功能
- **精确的 BPM 控制**：可在 30 到 300 BPM 之间调节
- **多种拍号**：支持各种拍号，包括 1/4、2/4、3/4、4/4、5/4、6/8、7/8、9/8 和 12/8
- **节拍细分**：全面的选项，包括基本细分（二分、四分、八分、十六分、三十二分音符）和连音（二连音、三连音、四连音、五连音、六连音、七连音、九连音）
- **声音选项**：多种声音类型，包括点击声、鼓声、木鱼声、电子音和金属音
- **音量控制**：分别控制重音和普通节拍的音量
- **视觉节拍指示器**：每个节拍的清晰视觉反馈

### 用户界面
- **现代设计**：时尚、专业的界面，带有渐变和流畅的动画效果
- **响应式布局**：自动适应任何设备屏幕尺寸
- **深色/浅色模式**：可在深色和浅色主题之间切换
- **直观控制**：易于使用的界面，带有清晰的视觉反馈

### 技术特性
- **Web Audio API**：高质量音频，低延迟
- **双重存储**：使用LocalStorage和Cookie实现持久化设置
- **浏览器兼容性**：支持现代浏览器和较旧版本
- **性能优化**：即使在较旧设备上也能流畅运行

## 📦 安装

要在本地运行此项目，请按照以下步骤操作：

1. **克隆仓库**：
   ```bash
   git clone <仓库地址>
   cd metronome-app
   ```

2. **安装依赖**：
   ```bash
   npm install
   ```

3. **启动开发服务器**：
   ```bash
   npm run dev
   ```

4. **构建生产版本**：
   ```bash
   npm run build
   ```

## 📖 使用指南

1. **开始/停止**：点击播放按钮开始节拍器，停止按钮暂停
2. **调整 BPM**：使用 BPM 滑块或输入字段设置所需的速度
3. **选择拍号**：从选项中选择适当的拍号
4. **设置节拍细分**：从全面的选项中选择所需的节拍细分，包括基本细分和连音
5. **选择声音**：为节拍器选择不同的声音类型
6. **调整音量**：使用音量滑块设置所需的音量级别
7. **切换深色/浅色模式**：根据偏好在深色和浅色主题之间切换
