功能模块 · FEATURES

丰富的 功能组合

九大核心模块，覆盖日常节拍训练的各类场景
01
精确节拍调度引擎
基于 Web Audio API 的音频调度算法，解决细分、摇摆等复杂节奏场景下的精准播放问题。支持 BPM 范围 20–300，覆盖从慢板到极速冲刺。
02
12 套拍号预设 + 自定义
4/4、3/4、6/8、9/8、2/2、3/2 等常用拍号一键切换，首拍自动加重音。支持自定义拍数与拍值，满足专业练习需求。
03
6 种细分 + Swing 摇摆
八分、十六分、三连音、五连音、二分音符等 6 种细分类型，Swing 摇摆强度可调，为爵士风格练习提供真实节奏感。
04
多音色系统 + 自定义频率
6 种预设音色（点击音、蜂鸣、木鱼、鼓点、明亮、温暖），支持自定义波形（方波/正弦/三角/锯齿）、频率与时长。
05
三级音量独立控制
重音拍、普通拍、细分音三档音量独立调节，帮助练习者更好地区分层级节拍结构。
06
主题配色与深色模式
12 套精心搭配的主题颜色 + 18 种自定义色 + 深色模式，适合长时间练习的护眼方案。
07
变速练习模板
慢速热身、稳定提速、高强度、高低变速、极限冲刺 5 种练习模板，配合自定义速度方案，循序渐进提升水平。
08
TAP 测速 + 预设 BPM
跟随歌曲节奏点击 TAP 按钮自动测算 BPM，8 种常用速度术语（Largo / Andante / Moderato / Presto 等）一键切换。
09
引导系统与体验优化
首次进入提供引导说明，键盘快捷键支持（空格播放、方向键调 BPM），localStorage 保存设置，Hash 路由二级页面导航。
创新亮点 · INNOVATION

值得关注的 设计与技术

不止是工具，更是体验
01
音频调度创新
结合 setTimeout 预调度 + AudioContext 时间戳精确对齐，解决移动端后台播放稳定问题，实现毫秒级精度的节拍播放。
02
视觉脉冲反馈
脉冲光环、BPM 数字缩放、节拍圆点高亮，三种视觉反馈方式叠加，练习时的节奏感知更加直观。
03
CSS 变量主题系统
每个主题通过独立的 CSS 变量定义，深色模式采用叠加覆盖策略，切换主题零延迟，扩展新主题仅需一行代码。
04
渐变视觉设计
渐变 BPM 数字卡片 + 径向光晕背景 + 伪元素滑块轨道，整体视觉现代、柔和且专业。
05
结构化变速练习
每段固定时长的 BPM 阶梯式推进，帮助练习者建立稳定的节奏记忆，从慢速热身到高强度冲刺系统训练。
06
零依赖纯前端
无需后端服务、无需外部 CDN、无需安装下载。打开即用，完全离线运行，所有设置在浏览器本地保存。
07
全端响应式
桌面、平板、手机全尺寸适配，支持触屏手势与键盘操作，刘海屏安全区处理，不同设备体验一致。
08
重音自定义标记
点击节拍圆点切换额外重音，支持任意复杂的重音排列，满足拉丁、非洲鼓等复杂节奏的练习需求。
