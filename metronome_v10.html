用户协议

使用条款与隐私政策

一、服务说明

节拍器 Pro 是一款免费的在线节拍器工具，为音乐学习者和从业者提供精准的节拍辅助功能。本应用所有功能均在本地浏览器中运行，无需服务器支持。

二、数据存储

本应用使用浏览器的 localStorage 存储您的个人设置（包括主题选择、BPM收藏、音量设置等）。这些数据仅保存在您的本地设备中，不会上传到任何服务器，我们不会收集任何个人信息。

三、音频权限

本应用需要使用浏览器的 Web Audio API 来播放节拍声音。首次使用时需要您与页面交互（如点击播放按钮）以激活音频功能，这是浏览器的安全策略要求。

四、免责声明

本应用按"现状"提供，不保证适用于所有场景。音频精度可能受到设备性能、浏览器实现和系统调度的影响。对于因使用本应用而产生的任何直接或间接损失，我们不承担责任。

五、AI 辅助开发说明

本应用由 AI 辅助开发完成，包括代码生成、界面设计和功能实现。AI 技术帮助提升了开发效率和产品质量，但最终产品经过了人工测试和优化。

六、使用建议

建议在安静环境下使用，音量适中以保护听力。长时间练习请注意休息。如果遇到音频延迟或不同步问题，建议关闭其他占用资源的应用或刷新页面。

七、协议更新

本协议可能会随版本更新而修订。继续使用即表示您同意修订后的协议条款。
