欢迎使用节拍器
一款完全本地运行、无网络依赖的精准节拍器，支持丰富的个性化设置与练习模板。
用户协议与隐私声明

使用说明

本应用为纯前端单文件应用，打开后即可使用，无需安装、无需注册账号。

数据与隐私

您的设置（BPM、主题、音色偏好等）保存在浏览器本地存储（localStorage）中，不会上传到任何服务器。清除浏览器数据或使用隐私浏览模式，这些设置将被清除。

权限说明

本应用仅需要音频播放能力，使用浏览器内置的 Web Audio API 合成节拍声音，不请求任何麦克风、摄像头或定位权限。

免责声明

本应用按"现状"提供，不提供任何明示或暗示的保证。因使用本应用造成的任何直接或间接损失，作者不承担任何责任。长时间使用耳机请适度降低音量，保护听力。
